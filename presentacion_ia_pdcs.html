2. Copiloto de Contenidos: Ingeniería de Prompts

Para rellenar masivamente los meses de Julio a Noviembre, usaremos IA. El secreto del éxito no es "hablarle" a la IA, sino darle un prompt restrictivo y técnico[cite: 11].

"Actúa como un diseñador curricular experto en el Subsistema de Educación Regular de Bolivia. Necesito redactar los campos para mi Spreadsheet de planificación de la asignatura de [Insertar Materia] para [Insertar Año de Escolaridad] correspondiente al mes de [Insertar Mes]. Dame el contenido estructurado en un bloque de texto separado por tabulaciones para copiar y pegar directamente, incluyendo: objetivos holísticos con las cuatro dimensiones, contenidos curriculares oficiales vigentes, orientaciones metodológicas detalladas (Práctica, Teoría, Valoración, Producción) y criterios de evaluación (Ser, Saber, Hacer)."[cite: 11]

Anatomía del Prompt (¿Por qué funciona?)

🎭 1. El Rol
"Actúa como diseñador curricular experto..." Obliga a la IA a usar lenguaje técnico pedagógico boliviano, evitando respuestas genéricas.
🎯 2. La Tarea
"Necesito redactar los campos para mi Spreadsheet..." Le da contexto de que los datos irán a una tabla, no a un documento de texto libre.
📋 3. El Formato
"Separado por tabulaciones para copiar y pegar..." Esto es vital. La tabulación permite que al pegar en Excel, cada dato caiga exactamente en su propia columna.
⚙️ 4. Restricciones
"Criterios de evaluación (Ser, Saber, Hacer)..." Nota para el Colegio Fielding: Recuerden instruir a la IA que el Saber vale 45 pts, Hacer 40 pts, Ser 10 pts y Autoevaluación 5 pts, omitiendo la dimensión del Decidir para alinear los PDCs con nuestro nuevo cuaderno pedagógico institucional.

3. Configuración del Motor Apps Script

Este código es el motor de nuestra fábrica. Lo insertaremos yendo al menú Extensiones -> Apps Script en nuestra hoja de cálculo[cite: 11].

function generarPDCsMasivos() { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = sheet.getDataRange().getValues(); var headers = data[0]; // Bucle para recorrer cada fila de docente (A partir de la fila 2)[cite: 11] for (var i = 1; i < data.length; i++) { var row = data[i]; var nombreArchivo = "PDC_" + row[7] + "_" + row[0]; // PDC_Mes_Materia[cite: 11] var doc = DocumentApp.create(nombreArchivo); var body = doc.getBody(); body.appendParagraph("PLAN DE DESARROLLO CURRICULAR (PDC)").setHeading(DocumentApp.ParagraphHeading.HEADING1); // Mapeo automático de llaves dinámicas {{columna}}[cite: 11] for (var j = 0; j < headers.length; j++) { body.appendParagraph(headers[j].toUpperCase() + ": " + row[j]); } doc.saveAndClose(); } }
🧠 ¿Qué hace este código? Lee fila por fila. Por cada fila, crea un Documento de Google nuevo, le pone de título el Mes y la Materia (ej. "PDC_Julio_Historia"), y vacía todo el contenido de las columnas dentro del documento[cite: 11].

4. Ejecución, Permisos y Producción Masiva

Al presionar "Ejecutar" por primera vez, Google nos lanzará una alerta de seguridad porque nuestro código va a crear archivos automáticamente. Debemos autorizarlo siguiendo esta ruta exacta[cite: 11]:

1️⃣

Revisar

Haga clic en el botón emergente que dice "Revisar permisos"[cite: 11].

2️⃣

Cuenta

Seleccione su cuenta institucional o de Gmail activa en la lista[cite: 11].

3️⃣

Avanzado

Ignore la alerta roja y pulse en las letras pequeñas que dicen "Avanzado" (abajo a la izquierda)[cite: 11].

4️⃣

Permitir

Haga clic en "Ir a Proyecto sin título (no seguro)" y finalmente presione el botón azul "Permitir"[cite: 11].

🚀 RESULTADO EN LA NUBE:
El script se ejecutará en segundo plano. Al dirigirse a su unidad de Google Drive, observará la aparición instantánea de todos sus documentos de PDC individuales listos para su impresión, descarga o envío a Dirección[cite: 11].
